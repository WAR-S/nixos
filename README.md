# nixos-nettop

## Сборка установочного ISO

На хосте с Nix (и включёнными flakes):

```bash
cd /path/to/nixos-nettop
nix build .#iso
```

Готовый образ будет в `result/iso/nixos-minimal-*.iso`. Запись на флешку:

```bash
# подставьте свой блок-устройство (lsblk)
sudo dd if=result/iso/nixos-minimal-*.iso of=/dev/sdX status=progress
sync
```

# Техническое задание

## Миграция инфраструктуры на NixOS с использованием слойной архитектуры

---

# 1. Цель проекта

Разработать воспроизводимую инфраструктуру на базе NixOS, обеспечивающую:

* декларативное описание инфраструктуры
* воспроизводимость конфигураций
* автоматическую сборку образов
* автоматическую установку системы
* управление контейнерной платформой
* GitOps-деплой приложений

Инфраструктура должна реализовываться с использованием **слойной архитектурной модели**.

---

# 2. Архитектурная модель

Система должна быть реализована в виде **наборa независимых слоев (layers)**.

Каждый слой:

* имеет собственную область ответственности
* реализуется как отдельный модуль конфигурации
* может переиспользоваться
* может быть отключен или расширен

Слои применяются последовательно.

## Общая схема слоев

```
Hardware Layer
      ↓
OS Layer
      ↓
Network Layer
      ↓
Security Layer
      ↓
Container Platform Layer
      ↓
Application Layer
```

---

# 3. Hardware Layer

## Назначение

Слой отвечает за взаимодействие операционной системы с аппаратным обеспечением.

## Функциональные требования

Должны быть реализованы:

* конфигурация устройств
* правила автоматического обнаружения устройств
* управление сетевыми интерфейсами
* поддержка модемов и сетевых устройств

## Компоненты

* правила устройств через udev
* поддержка:

  * USB modem
  * Ethernet
  * WiFi адаптеров

## Функции

* автоматическое определение типа сетевого подключения
* автоматическая активация соответствующих сервисов

---

# 4. OS Layer

## Назначение

Слой отвечает за базовую конфигурацию операционной системы.

## Функциональные требования

Слой должен обеспечивать:

* настройку базовой системы
* конфигурацию systemd сервисов
* управление пакетами
* управление пользователями
* конфигурацию bootloader

## Компоненты

* системные пакеты
* базовые сервисы
* управление пользователями
* файловые системы
* системные параметры ядра
